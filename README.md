# Macropad
A 9-key macropad that includes 2 rotary encoders. I made this project to challenge myself so that I can be better prepared to create real-world solutions and as a starting point for more future advanced projects, such as designing my own miniature fan in the shape of a plane engine. It is designed as a compact, programmable input device using QMK firmware, suitable for shortcuts, media control, and custom workflows.
